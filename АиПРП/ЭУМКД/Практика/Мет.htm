Методические указания к выполнению контрольных работ №1,  №2.

 

1.     Для выполнения контрольной работы №1 нужно освоить материал раздела 3 (темы 3.1, 3.3) , а также лабораторной работы по теме 3.3.

2.     В качестве литературных источников можно рекомендовать [1-3,7] основного списка.

3.     В отчете по контрольной работе осветить указанные в задании теоретические вопросы. Разобрать программную реализацию предложенного для выполнения задания.

 

4.      Для выполнения контрольной работы №2 нужно освоить материал разделов 1.2, 1.4, 2.4 а также лабораторную работу по теме 2.4 и 3.3.

5.     В качестве литературных источников можно рекомендовать [1-3] основного списка.

6.     В отчете по контрольной работе осветить указанные в задании теоретические вопросы. Разобрать программную реализацию предложенного для выполнения задания.

7.     Номер варианта определить как остаток от деления номера зачетной книжки на 17. Если в остатке получится число, превосходящее номер вариантов, то повторить процедуру разделив номер зачетной книжки на 13.
