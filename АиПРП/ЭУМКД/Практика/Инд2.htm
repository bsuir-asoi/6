ИНДИВИДУАЛЬНАЯ ПРАКТИЧЕСКАЯ РАБОТА № 2.

ПЕРСИСТЕНТНЫЕ КЛАССЫ.

 

Цель практической работы.

Работа посвящена технологии Hibernate – одной из важных технологий работы с внешними данными в Java EE.

Требуется написать приложение, демонстрирующее работу персистентного класса. Приложение должно иллюстрировать добавление и удаление записей из базы данных, а также их вывод. Реализовать клиентскую часть в виде окна с текстовыми полями для вывода записи из таблицы базы данных.

 

Базу данных реализовать либо в MySql, либо в Derby, либо в Access. База данных должна располагаться на стороне сервера (GlassFish, Tomcat, JBoss).

Результаты работы оформить в виде отчета с приведением текста программ и необходимых комментариев.

В результате выполнения работы студент должен освоить технику привязки физической таблицы базы данных к классу Java, структура полей которого соответствует физической таблице, а также реализовать методы доступа к полям записей, их добавлению и удалению из базы.
